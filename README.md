这里是GTM（Graduate Texts in Mathematics）系列丛书的展柜

Graduate Texts in Mathematics（GTM）是Springer-Verlag公司出版的一个数学基础系列书籍，包括了数学分析、高等代数、复变分析、概率论、随机过程等较高层次的优秀数学教材。

我们把GTM系列丛书的电子版上传到了这里，浅川科學大學和唐小菲大学的小菲们可以按需下载
