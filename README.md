# 代数学
这里是所有代数学教材

代数是数学的一个分支，它使用字母而不是数字来研究方程求解。 现代代数极大地扩展了其研究范围，成为一门研究代数系统（如半群、群、环、多环（代数）、交换场和束等代数系统的学科（抽象代数）。 代数的概念也渗透到分析和几何学中，并在为数学的各个领域提供通用语言方面发挥着作用。

出现在以下代数字段名称中的半群、群、环、多环（代数）、字段和束是典型的代数结构。

群、环、多环和场理论起源于 Evaristo Galois 等人对代数方程解的研究，束理论起源于 George Boole 的逻辑数学研究。

在现代日本大学中，线性代数在第一年和第二年与微积分一起学习，但线性代数是代数的一个分支，研究称为向量空间的代数系统。
